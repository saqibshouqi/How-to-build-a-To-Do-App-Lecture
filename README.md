# Anuha's To Do App

Simple single-file Anuha's To Do App (vanilla JS) that persists data in `localStorage`.

How to run

- Open `index.html` in your browser (double-click or use Live Server).

Features

- Add, edit (double-click or Edit), delete todos
- Mark complete/uncomplete
- Set task progress with a slider and view progress bars
- Default task completion percentage is 0%
- Filter: All / Active / Completed
- Clear completed
- Persistent storage via `localStorage`
